Tool 1 - CSS Foundation

  • Sassquatch
  • About HTML semantics and front-end architecture
  • Class name semantics are arbitrary - except for the people working with the code...
  • Embrace document flow by setting useful base styles.

Be explicit, not implicit

// clever
.unit .docsection:first-child {
  padding-top: $spacing;
  border-top: 0;
}

// better
.docsection--boxhead {
  padding-top: $spacing;
  border-top: 0;
}
